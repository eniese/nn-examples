# Sample Neural Network code

This project contains a presentation about AI, Neural Networks
and other things. This presentation was held at an OSP company
evening to spark a debate about AI and how we should adapt/adopt
this booming development into our company and skill set.

The presentation: AI friend or foe?.pdf

AI friend or foe?.key is an Apple Keynote source that is used
to create the PDF

NeuralNetwork.eddx => the NN diagrams used in the presentation (Wondershare EdrawMax)

nn.py & train_nn.py:
Some Python3 code to train and run a very simple Neural Network that can classify as animal based
on the input (number of legs).
It knows two kinds of animals: dogs & spiders.

train_nn.py -> trains this NN and gives weights & biases
nn.py -> implementation of this simple NN

This uses just the standard library of python3 so you don't
need any dependencies
